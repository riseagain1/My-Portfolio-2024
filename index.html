Morse Code Fight

"Morse Code Fight" is an innovative game concept where players use Morse code to control their characters in a fighting or action game. Utilizing a single button, players input commands with short and long presses corresponding to dots and dashes of Morse code. This unique control scheme not only adds a strategic depth to the gameplay but also provides an educational aspect as players learn Morse code.
